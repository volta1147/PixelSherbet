### 기본 기능
`:>ping` : 기본적인 속도 테스트
`:>echo` : 메시지 출력하기
`:>info` : 봇 정보 띄우기
`/help` : 이거 띄우는거

### Botplus
`:>print` : 원본 메시지를 지우고 출력하기, 답장 지원됨
`:>reaction` : 다른 글에 반응 달기
`:>random` : 항목 여러개 중에 하나 답하기
`/slashprint` : 슬래시 명령어로 출력하기

### TTS
`:>tts_chat speak` : tts로 출력하기
`/tts speak` : tts로 출력하기
`/tts exit` : tts봇을 음성채팅에서 내보내기
