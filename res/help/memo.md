# 도움말
## 기능 설명
### Memo
:> 명령어
* `:>memo open` : 메모 출력
* `:>memo edit` : 메모 작성/편집
* `:>profile` : 자기소개 출력
* `:>introduce` : 자기소개 편집
* `:>rev memo` : 메모의 구버전 열기
* `:>rev profile` : 상대방의 자기소개 구버전 열기
* `:>rev myprofile` : 본인의 자기소개 구버전 열기
* `:>back memo` : 메모 되돌리기
슬래시 명령어
* `memo open` : 메모 출력
* `memo edit` : 메모 수정
* `memo append` : 메모 이어쓰기
* `memoui menu` : 메뉴 불러오기
