# 도움말
## 기능 설명
### Memo
메모
* `/memo open` : 메모 출력
* `/memo edit` : 메모 수정
* `/memo random` : 아무 줄이나 긁어오기
* `/memo search` : 메모 검색 (제목)

서버 내 메모 (다른 메모와 호환되지 않습니다!)
* `/private open` : 메모 출력
* `/private edit` : 메모 수정
* `/private random` : 아무 줄이나 긁어오기
* `/private search` : 메모 검색 (제목)

자기소개
* `/profile open` : 메모 출력
* `/profile edit` : 메모 수정
